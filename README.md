Dies ist ein Deadlocks Simulator.
Es simuliert mehrere deadlocks dann versucht sie mit einer der 3 beinhaltene policies zu beheben/verhindern/vermeiden.
Damit der GUI funktioniert muss das folgende Programm installiert werden: https://www.msys2.org/ auf windows,
in Ubunto oder WSL muss das folgende command in terminal gegeben werden: sudo apt-get install libgtk-3-dev
