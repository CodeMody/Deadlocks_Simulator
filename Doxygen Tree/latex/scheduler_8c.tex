\doxysubsection{Detailed Description}
Implementierung des ereignisgesteuerten Simulator-\/\+Kerns. 

Der \doxylink{struct_scheduler}{Scheduler} ist die zentrale Steuereinheit der Simulation. Er verwaltet eine zeitgeordnete Event-\/\+Queue (Min-\/\+Heap) und verarbeitet Ereignisse chronologisch. Ressourcenanforderungen werden an die aktive \doxylink{struct_policy}{Policy} delegiert; abgelehnte Anfragen werden automatisch für den nächsten Tick erneut eingeplant, bis das Retry-\/\+Limit ({\ttfamily MAX\+\_\+\+RETRIES}) erreicht ist.

Die interne Struktur ({\ttfamily struct} \doxylink{struct_scheduler}{Scheduler}) ist nach dem Opaque-\/\+Pointer-\/ Muster versteckt, sodass externe Module nur über die öffentliche API aus {\ttfamily \doxylink{scheduler_8h}{scheduler.\+h}} mit dem \doxylink{struct_scheduler}{Scheduler} interagieren können.
