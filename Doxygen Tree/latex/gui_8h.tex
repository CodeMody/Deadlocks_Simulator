\doxysubsection{Detailed Description}
Öffentliches Interface für das GTK-\/3-\/\+Frontend des Deadlock-\/\+Simulators. 

Dieses Modul implementiert die grafische Benutzeroberfläche mit folgenden Elementen und Funktionen:\+
\begin{DoxyItemize}
\item Drei Policy-\/\+Schaltflächen zur Auswahl der Deadlock-\/\+Behandlungsstrategie.
\item Eine Step-\/\+Schaltfläche für manuelle Einzelschritte der Simulation.
\item Eine Statusleiste mit aktivem Policy-\/\+Namen, Verschiebungs-\/\+Zähler und Tick.
\item Ein scrollbares Textfenster für den Echtzeit-\/\+Snapshot des Systemzustands.
\item Eine Legende zur Erklärung der angezeigten Felder.
\item Automatisches Update der Anzeige einmal pro Sekunde via GTK-\/\+Timer. 
\end{DoxyItemize}
