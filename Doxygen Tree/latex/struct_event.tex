\doxysubsection{Detailed Description}
Einzelnes Ereignis in der zeitgesteuerten Event-\/\+Queue. 

Ereignisse werden in einem Min-\/\+Heap nach {\ttfamily time} sortiert. Bei abgelehnten Anfragen wird das Ereignis mit erhöhtem {\ttfamily retries-\/\+Zähler} erneut in die Queue eingetragen.
