\doxysubsection{Detailed Description}
Einstiegspunkt des Deadlock-\/\+Simulators. 

Initialisiert das GTK-\/3-\/\+Framework, erstellt das Hauptfenster der grafischen Benutzeroberfläche und startet die GTK-\/\+Ereignisschleife. Die Simulation selbst beginnt erst, wenn der Benutzer über die GUI eine der drei Deadlock-\/\+Policies auswählt.
