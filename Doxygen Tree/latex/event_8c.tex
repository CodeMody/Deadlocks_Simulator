\doxysubsection{Detailed Description}
Implementierung des Min-\/\+Heap-\/basierten Ereignis-\/\+Heaps. 

Der {\ttfamily \doxylink{struct_event_heap}{Event\+Heap}} ist eine dynamisch wachsende Prioritätswarteschlange, die Ereignisse aufsteigend nach ihrem logischen Zeitstempel ({\ttfamily time}) sortiert. Er verwendet 1-\/basierte Indizierung, sodass der Index 0 unbenutzt bleibt und die Eltern-\/\+Kind-\/\+Beziehungen durch einfache Bit-\/\+Operationen berechnet werden können.

Zur Berechnung der Heap-\/\+Positionen dienen folgende Makros:\+
\begin{DoxyItemize}
\item {\ttfamily \doxylink{event_8c_a25d5ae2e52c533b5f27895fcf7a4ea95}{LEFT(i)}} = linkes Kind = i × 2
\item {\ttfamily \doxylink{event_8c_a134d4f19a207929c1e9716743771ae5a}{RIGHT(i)}} = rechtes Kind = i × 2 + 1
\item {\ttfamily \doxylink{event_8c_a91004761621b8ef1c66f3601988fde7c}{PARENT(i)}} = Elternknoten = i /\+ 2 
\end{DoxyItemize}
