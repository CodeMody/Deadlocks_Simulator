Detailed Description

Implementierung des Min-Heap-basierten Ereignis-Heaps.

Der EventHeap ist eine dynamisch wachsende Prioritätswarteschlange, die Ereignisse aufsteigend nach ihrem logischen Zeitstempel (time) sortiert. Er verwendet 1-basierte Indizierung, sodass der Index 0 unbenutzt bleibt und die Eltern-Kind-Beziehungen durch einfache Bit-Operationen berechnet werden können.

Zur Berechnung der Heap-Positionen dienen folgende Makros:

  • LEFT(i) = linkes Kind = i × 2
  • RIGHT(i) = rechtes Kind = i × 2 + 1
  • PARENT(i) = Elternknoten = i / 2
