Detailed Description

Implementierung der Systemzustandsverwaltung.

Dieses Modul verwaltet den zentralen SystemState, der das Ressourcensystem nach dem Bankier-Modell abbildet. Es stellt Funktionen zur Allokation, Manipulation und Analyse des Zustands bereit. Der Kern des Moduls ist die Implementierung des Bankier-Algorithmus (state_is_safe()), der prüft, ob eine sichere Abarbeitungsreihenfolge aller Prozesse existiert.

◆ state_is_safe()

bool state_is_safe ( const SystemState * st)

Prüft mit dem Bankier-Algorithmus, ob der aktuelle Zustand sicher ist.

Initialisiert den Arbeitsvektor work mit den aktuell verfügbaren Ressourcen. In einer Schleife werden alle noch nicht abgeschlossenen Prozesse gesucht, deren Bedarf durch work gedeckt werden kann. Ein solcher Prozess wird als "fertig" markiert und seine Ressourcen werden zu work addiert. Die Schleife wird wiederholt, bis kein Fortschritt mehr möglich ist. Konnten alle Prozesse abgeschlossen werden, ist der Zustand sicher.

Parameters
stZeiger auf den zu prüfenden Systemzustand (const).
Returns
true, wenn ein sicherer Ablaufplan existiert; sonst false.
