Detailed Description

Einzelnes Ereignis in der zeitgesteuerten Event-Queue.

Ereignisse werden in einem Min-Heap nach time sortiert. Bei abgelehnten Anfragen wird das Ereignis mit erhöhtem retries-Zähler erneut in die Queue eingetragen.
