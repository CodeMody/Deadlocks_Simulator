Detailed Description

Gemeinsame Typdefinitionen für den Deadlock-Simulator.

Diese Datei definiert alle grundlegenden Datenstrukturen und Typen, die von den Modulen core und policy gemeinsam genutzt werden. Dazu gehören Ereignistypen, die Ereignisstruktur, der Systemzustand sowie Funktionszeiger für das Policy-Interface.

◆ policy_cleanup_f

typedef void(* policy_cleanup_f) (struct Policy *p)

Funktionszeiger-Typ für die Aufräumfunktion einer Policy.

Gibt den gesamten dynamisch allokierten Speicher der Policy frei, einschließlich des privaten Kontexts und der Policy-Struktur selbst.

Parameters
pZeiger auf das freizugebende Policy-Objekt.

◆ policy_request_f

typedef bool(* policy_request_f) (struct Policy *p, struct SystemState *st, const Event *ev)

Funktionszeiger-Typ für den Request-Callback einer Policy.

Wird vom Scheduler aufgerufen, wenn ein EV_REQUEST-Ereignis verarbeitet wird. Die Funktion entscheidet, ob die Anfrage genehmigt wird.

Parameters
pZeiger auf das Policy-Objekt (inkl. privatem Kontext).
stAktueller Systemzustand.
evDas zu bearbeitende Anforderungsereignis.
Returns
true, wenn die Anfrage genehmigt werden soll; sonst false.

◆ policy_tick_f

typedef void(* policy_tick_f) (struct Policy *p, struct SystemState *st, uint64_t now)

Funktionszeiger-Typ für den periodischen Tick-Callback einer Policy.

Wird nach jedem verarbeiteten Ereignis aufgerufen und ermöglicht der Policy periodische Wartungsaufgaben (z. B. Deadlock-Erkennung).

Parameters
pZeiger auf das Policy-Objekt.
stAktueller Systemzustand.
nowAktuelle Simulationszeit.

◆ EventType

enum EventType

Aufzählung aller möglichen Ereignistypen in der Simulation.

Jedes Ereignis in der Event-Queue besitzt einen dieser Typen, der bestimmt, wie der Scheduler das Ereignis verarbeitet.

Enumerator
EV_REQUEST 

Ein Prozess fordert Ressourcen einer Klasse an.

EV_RELEASE 

Ein Prozess gibt zuvor gehaltene Ressourcen frei.

EV_CHECKPOINT 

Prüfereignis; kann für Zustandstests genutzt werden.

EV_TERMINATE 

Ein Prozess beendet sich und gibt alle Ressourcen frei.
